Descripción del problema:

En esta tarea se busca implementar una interfaz grafica para los elementos usados en la tarea n°1. Para ello se utilizaran coponentes 
graficos los cuales interaccionarán con el funcionamiento del dispositivo mouse y teclado.



Etapa4:

	-BallView.java

	-FixedHookView.java

	-LabMenuListener.java
	-MouseListener.java
	-MyWorld.java
	-MyWorldView.java
	-PhysicsElement.java
	-PhysicsLab.java
	-Simulateable.java
	-Spring.java
	-SpringAttachable.java
	-SpringView.java

Para compilar y ejecutar los archivos de la etapa 4 se deben 
seguir los siguientes pasos, estando situados en la carpeta
correspondiente a los codigos:

1.- En terminal ejecutar comando make, para compilar y generar 
    archivos .class.
2.- Para ejecutar el programa se debe tipear en el terminal
    make run.
3.- Si desea borrar los archivos .class, solo debe tipear
    make clean.
4.-Si desea ver la documentación se debe tipear make doc , lo cual creará la documentacion en una nueva carpeta dentro del directorio 
   donde contenga el makefiel llamada documentation, en la cual se encontrarán los archivos html correspondientes.
    
Nota: 1-Al presionar la tecla "n" los elementos fisicos tales como bolas, resortes, etc, cambiaran de su estado, el color.
