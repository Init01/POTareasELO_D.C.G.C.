Descripción del problema:

En esta tarea se busca implementar a la interfaz grafica de la tarea numero 2, graficos que muestren la energia cinetica, potencial y total del sistema.
Ademas se introducirá efecto de sonido para la colision de las bolas y tambien la modificación de esta aplicacion para que logre ejecutarse en navegadores web.

Para compilar y ejecutar los archivos de la etapa 4 se deben 
seguir los siguientes pasos, estando situados en la carpeta
correspondiente a los codigos:

1.- En terminal ejecutar comando make, para compilar y generar 
    archivos .class.
2.- Para ejecutar el programa se debe tipear en el terminal
    make run.
3.- Si desea borrar los archivos .class, solo debe tipear
    make clean.
4.- Si desea crear un archivo jar , se debe tiper make jar
5.- Si desea ejecutar el programa con Appletviewer se debe tipear make runApplet
